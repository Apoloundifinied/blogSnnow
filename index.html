ApoloCpf Python

Verificador de CPF feito em Python. Simples, funcional e disponível como pacote npm para fácil integração.

📌 Sobre
Este é um simples e funcional validador de CPF feito em JavaScript, agora disponível como pacote npm para fácil integração em seus projetos.

🚀 Como Usar 
const { validaCPF } = require('@snnowip/text-tools')

// Exemplo de uso
console.log(validaCPF('123.456.789-09')) // false
console.log(validaCPF('935.411.347-80')) // true
Instalando via npm 
npm install @snnowip/text-tools

Consul ApiMonitorUsr Node.js

Ferramenta minimalista e pronta para produção para monitoramento de APIs no terminal, feita em Node.js.
Dashboard TUI, métricas detalhadas, notificações por e-mail, design em escala de cinza e animações ASCII.

✨ Features & Instalação
  • Monitoramento em tempo real com node-cron
  • Dashboard TUI com blessed
  • Métricas detalhadas: uptime, tempo de resposta, falhas
  • Alertas por e-mail via nodemailer
  • Histórico persistente em status.json
  • Exponential Backoff para APIs com falha
  • Design minimalista com animações ASCII
  • Configuração via CLI (yargs)
  • Arquitetura extensível e robusta
📦 Instalação 
git clone https://github.com/rbphishing/apimonitorusr.git
cd monitor-sys/monitoramento-terminal/Utils
npm install axios inquirer chalk node-cron nodemailer blessed yargs
# Crie urls.json e config.json conforme README
node interface.js
